Teach AI - Assistente Educacional Inteligente

Formação Téc. (T.I.)

O Teach AI é uma plataforma desenvolvida para combater a desinformação escolar e auxiliar alunos com dúvidas fora da sala de aula. O sistema utiliza Inteligência Artificial para funcionar como um tutor virtual seguro: ele não inventa respostas, mas consulta exclusivamente o material didático (PDFs) fornecido pela escola para gerar explicações precisas. O projeto foi concebido para resolver o problema de fontes não confiáveis na internet e a sobrecarga dos professores, entregando um MVP (Produto Mínimo Viável) funcional e validado tecnicamente.

Detalhes do Projeto

  • Tecnologias: O Backend foi construído em Python com o framework Flask. Para o Frontend, utilizamos HTML5, JavaScript e Tailwind CSS. O coração da inteligência é o modelo Google Gemini 2.5 Flash, integrado a um banco de dados vetorial FAISS e Sentence Transformers (SBERT) .
  • Desafios encontrados: Inicialmente, tentamos utilizar IAs locais (como Ollama/Mistral), mas enfrentamos limitações de hardware e suporte na infraestrutura disponível. A solução foi migrar para uma arquitetura em nuvem utilizando a API gratuita do Google Gemini, o que garantiu performance sem custos.
  • Funcionalidades implementadas: Implementamos a tecnologia RAG (Retrieval-Augmented Generation) para contexto, um sistema de Dupla Verificação (onde a IA revisa a própria resposta antes de enviar) e um painel administrativo para upload de PDFs escolares (Base de Conhecimento).
